# VoxelDollarPage

Buy 1 voxel or a block of voxels in a 100×100×100 world (1,000,000 voxels total).

## How it works
1. Open the viewer and drag to select a block.
2. Pay via Revolut: https://revolut.me/francelqup (use the note shown in the UI).
3. Fill the submission form (logo + URL + tooltip).
4. After manual approval, your block appears as SOLD in the viewer.

## Files
- index.html: viewer UI
- app.js: Three.js scene + drag-box selection + Revolut flow
- voxel.js: helper functions (grid math, selection → block, rendering sold blocks)
- sold.json: list of sold blocks
- buy.html: submission form
